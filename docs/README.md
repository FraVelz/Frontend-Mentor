# Documentación del repositorio

Archivos de apoyo (plantillas, convenciones e instrucciones). **No** se publican en GitHub Pages como rutas del sitio: viven en `docs/` solo para el trabajo en el repo y para que humanos/IA tengan rutas estables.

| Archivo                                                    | Uso                                                                         |
| ---------------------------------------------------------- | --------------------------------------------------------------------------- |
| [`readme-template.md`](./readme-template.md)               | Plantilla para generar `nombre-reto/README.md` al incorporar un reto.       |
| [`Instructions_IA.md`](./Instructions_IA.md)               | Flujo para actualizar el índice, README y convenciones al añadir retos.     |
| [`NOTAS-TECNICAS.md`](./NOTAS-TECNICAS.md)                 | Estructura del repo, Pages, despliegue y desarrollo local.                  |
| [`CONVENCIONES-METADATOS.md`](./CONVENCIONES-METADATOS.md) | Metadatos en `<head>` (SEO, Open Graph, Twitter) del índice y de cada reto. |

En el [`README.md`](../README.md) de la raíz hay enlaces a notas técnicas y, cuando aplique, a esta carpeta.
