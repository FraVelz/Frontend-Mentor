# Frontend Mentor challenges

_[Versión en español](./README.md)_

This repository is a **collection of solutions** to [Frontend Mentor](https://www.frontendmentor.io) challenges: exercises with real designs to practice layout and front-end work, from a design to a working page.

## Part 1 and part 2

- **Part 1 — this repository (vanilla):** meant to **strengthen and practice HTML, CSS, JavaScript, and Tailwind CSS** in the browser, one folder per challenge, with **no mandatory build step**; a good fit when you want to master layout, the cascade, accessibility, and basic JS.
- **Part 2 — Next.js (more advanced):** the same line of challenges with a workflow closer to a professional one (routing, React components, data, deployment). The repository is here: **https://github.com/FraVelz/Frontend-Mentor-Nextjs**.

> [!NOTE]
> Each challenge lives in **its own folder**; when the site is published, you can open them one by one from the index at the repository root.

## Viewing the challenges in the browser

The project is deployed on **GitHub Pages**; the home page (`/`) lists each challenge with links and a screenshot for each section.
